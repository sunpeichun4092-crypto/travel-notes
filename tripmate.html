🌊 TripMate · 旅程伴侣

3-8 人小团体出游，从「行前一起规划」到「行中省心管账」再到「行后留下回忆」。

为小团体而生 · 一站式陪伴
一个完整的 monorepo：React Native (Expo) 移动端 + Express 后端 + Supabase (Postgres / Auth / Storage / RLS) + OpenAI GPT-4o Vision AI 游记。

✨ MVP 包含的五个模块

每个模块独立可用，组合起来覆盖小团体出游全生命周期。

🧭

发现目的地

5 步 Wizard 收集团队偏好，GPT + 实时网页搜索（SerpAPI / Bing）混合输出 5-7 个候选。

  • 基地 / 预算 / 风格 / 避雷 / 时间 / 住宿
  • 小众等级 1-5、风险等级 1-5
  • 当地人 tips、参考来源链接
🗺

行程协作

一键创建旅程，邀请伙伴加入并共同维护行程。

  • 6 位邀请码加入
  • 成员头像列表、行程倒计时
  • 子页面 tile 入口
💰

团队记账

多人分账，谁付谁还一目了然。

  • 等额 / 加权两种分账方式
  • 智能净额计算、最少转账方案
  • 整数分计算无浮点误差
📸

共享相册

所有成员的照片汇集到同一个相册，按拍摄日期自动梳理。

  • Expo ImagePicker 多选上传
  • 签名 URL 直传 Storage
  • 按拍摄日期分组、点赞
✈️

AI 游记

GPT-4o Vision 多图理解，自动整理成结构化游记。

  • 多图理解、按时间线整合
  • 结构化 JSON：开篇 / 每日篇 / 收尾
  • 可保存回看，导出分享

🏗️ 技术架构

Monorepo 设计，各部分独立部署，依赖关系清晰。

移动端
React Native + Expo
iOS / Android 跨端，Expo 托管开发流，支持 OTA 更新。
后端 API
Node.js + Express
轻量 API gateway，主要做鉴权透传与第三方服务编排。
数据库
Supabase Postgres
行级安全 RLS 保证团队间数据隔离。
认证
Supabase Auth
手机号 / 邮箱登录，JWT 透传至 RLS。
对象存储
Supabase Storage
签名 URL 直传，规避后端带宽。
AI
OpenAI GPT-4o Vision
多模态推理 + 实时网页搜索（SerpAPI / Bing）。

🛤️ 旅程生命周期

每个阶段都有对应的模块陪伴。

行前

一起规划

发现目的地 Wizard + 行程协作邀请伙伴，预算和风格偏好一次说清。

行中

省心管账

团队记账实时净额，谁该转给谁一键发出，照片随手丢进共享相册。

行后

留下回忆

AI 游记一键整理多日图文，导出结构化游记保存或分享朋友圈。
